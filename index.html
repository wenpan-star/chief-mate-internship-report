见习心得 · 职责解析
专题一：船上见习的基本情况、收获与体会

从20**年**月**日上船见习至今近三个月，在船长的领导和大副的培训指导下我进度很快，同时也体会到了作为一名部门长的责任与担当，以下几点是我三个月的收获与心得：

大副是甲板部的部门长，是船长的主要助手，要做好大副，首先要深刻认识和了解大副的职责和任务。大副除了完成0400~0800/1600~2000的航行值班外，还需完成以下任务：

  • 领导甲板部全部工作，协助船长负责甲板部安全生产，主持安全活动，督促二副三副做好本职工作，保证本部门安全管理体系有效运行；负责甲板部船员及实习生技术业务培训；负责垃圾记录簿、货物记录簿、油类记录簿的正确记载和保管，以及甲板部其他图纸、资料文件的保管；负责甲板部人员考勤、值班等安排。
  • 主管船舶货物运输，全面负责货物配载、装载、卸载、交接和运输管理，保证货舱适货、合理积载，正确批注大副收据，安全装卸和保管货物，保证无货差、货损。
  • 根据本船特性和货物数量进行合理配载、压载，满足船舶稳性、强度及吃水差要求，进行有效安全作业，保障货物和船舶安全。
  • 熟悉船体结构、甲板设备性能及操作方法，负责维护船体、货舱装卸设备、阀门、救生消防及应急设备，使其随时可用并做好记录。
  • 负责淡水舱、污水舱、压载舱的管理、测量并记录；在船长授权下负责压载水管理，遵守国际法规及港口规则，制作压载水管理计划报船长审批；指示水手长进行压载舱、淡水舱、货舱的注排水工作并记录；负责淡水储量和消耗管理，拟定节水措施和应急供水方法，报船长批准后实施，按规定进行压载水置换、消毒等工作。
  • 编制甲板部修船项目单，领导厂修时甲板部监修和自修项目，编制年度、季度和航次维修保养计划，交船长审批后执行。
  • 督促甲板部做好开航前和到港前准备工作，进出港靠离泊在船首值守，指挥系解缆及抛起锚工作。
  • 督促相关人员保证救生消防设备、堵漏器材和防污器材处于良好状态；在各应急演习中担任（或协助轮机长）现场指挥，并将演习内容记录航海日志。
  • 负责甲板部物料管理，监督备件、物料、工具、劳保用品的申领、盘点、检查、验收、保管、使用及报销，按规定上报甲板部物料消耗、库存数量。
  • 当船上未配备专职医生时，负责全船医疗工作，管理医务室，保持医疗器械完好，检查药品有效期，按麻醉品规定保管、使用和年报；负责生病船员急救、医护，填写《船员就医登记本》，发现危急病情及时报告船长，发现传染病采取有效隔离及预防措施。
  • 防止甲板部任何形式的海洋污染事故，根据《MARPOL公约》和相关管理计划要求，主管船舶垃圾收集、存放和处理，按规定填写《船舶垃圾记录簿》、《油类记录簿》、《货物记录簿》并报告船长签阅；向垃圾接收船或岸上排放垃圾时应向接收单位索要收据并妥善保管。
  • 担任船舶保安员，在船长授权和领导下主管船舶保安事宜，负责船舶保安系统测试管理，全面开展船舶保安巡查、船舶保安演习现场指挥、船员培训训练，保障船舶安全。
  • 当船长因故不能履行职责时，临时代理船长职务，开展船舶工作，保证船舶、人员和货物安全。

大副是甲板部的部门长，是管理级高级船员。安全航行、货物操作仅为其工作的一方面，更重要的是部门综合管理（安全管理）。管理的好坏直接影响甲板部工作效率及运输任务的优质完成，也是衡量大副工作好坏的重要指标。大副的管理职责包括但不限于：

大副的良好监控和管理是甲板部安全有效运行的重要保证。做好大副必须加强业务学习，不断提高监控管理能力，以公约标准为目标，向国际化方向努力。中国船员的薄弱点是英语，因此要加强英语学习，特别是英语口语和沟通能力，并严格要求甲板部其他人员努力学习，为更好走向世界市场而努力。

航行规则 · 值班程序
专题二：确立值班安排和程序
《1972年国际海上避碰规则》的掌握

《1972年国际海上避碰规则》是为确保船舶航行安全、预防和减少船舶碰撞，规定在公海和连接公海的一切通航水域共同遵守的海上交通规则。规则主要包含定义、号灯号型、驾驶及航行规则等，对号灯位置和颜色、失控船舶号灯号型、各种能见度航行及驾驶规则等作了详细规定。规则自1977年生效以来，经多次修正（1981、1987、1989、1993、2001、2007年）。经修正后的规则包括以下部分：

（一）适用范围与定义
1. 适用于公海及连接公海可供海船航行的一切水域中的一切船舶。
2. 不妨碍主管机关对港外锚地、港口、江河、湖泊等特殊规定。
3. 不妨碍各国政府对军舰及护航船舶、结队捕鱼渔船制定的额外信号规定。
4. 组织可采纳分道通航制度。
5. 对特殊结构船舶如不完全符合条款……
6. 定义了机动船、帆船、捕鱼船舶、水上飞机、失控船、操纵能力受限船、限于吃水船、在航船长度宽度、互见、能见度不良、地效翼船等。

（二）号灯号型及声响信号
号灯号型在各种天气中均应遵守，号灯显示时间为日落到日出及能见度不良时，号型在白天应遵守。长度≥12m船舶应配备号笛；≥20m应配备号笛、号钟；≥100m应配备号笛、号钟、号锣。规则规定了各种能见度中互见及非互见情况下的不同声响信号及灯光信号补充。指引信号不应被误认为其他信号，避免使用频闪灯等高亮度间歇灯。

（三）任何能见度下的行动规则
1. 瞭望：每一船应使用视觉、听觉及一切有效手段保持正规瞭望。
2. 安全航速：考虑能见度、通航密度、船舶操纵性能、夜间背景亮光、风浪流及靠近危险物、吃水与水深关系、雷达特性等。
3. 碰撞危险：使用一切有效手段断定，如有怀疑应认为存在危险。
4. 避免碰撞的行动：及早、大幅度行动，运用良好船艺，避免一连串小变动。
5. 狭水道：船舶应尽量靠近右舷水道内安全航行，捕鱼船不应妨碍他船，穿越应避免妨碍。
6. 分道通航：在通航分道内顺着总流向行驶，尽可能让开分隔线，端部驶入/驶出角度尽量小，穿越时与总流向成直角。

（四）船舶在互见中的行动规则
1. 对遇局面：两机动船相反航向构成碰撞危险时各自向右转向。
2. 交叉相遇：有他船在本船右舷时应给其让路，避免横越他船船首。
3. 让路船行动：及早采取大幅行动，宽裕让清。
4. 直航船行动：当让路船未按规定行动时，可独自采取操纵行动避免碰撞；逼近到单凭让路船行动不能避免碰撞时，也应采取最有利避碰行动。
5. 船舶之间责任：机动船给失控船、操限船、捕鱼船、帆船让路；帆船给失控船、操限船、捕鱼船让路；捕鱼船尽可能给失控船、操限船让路；避免妨碍限于吃水船安全通行；水上飞机应宽裕让清所有船舶。

（五）能见度不良时的行动规则
适用于不在互见中的船舶。以安全航速行驶，机动船将机器备妥；仅凭雷达测到他船时应判定是否形成紧迫局面，尽早采取避让行动；听到他船雾号在本船正横以前或不能避免紧迫局面时，应减至维持航向的最小速度，必要时完全停住，直至危险过去。

值班安排与航行值班

值班安排：值班编制必须适应环境和情况，考虑驾驶台始终有人值守、天气能见度、航行危险、助航仪器、自动舵、特殊操作环境等因素。值班制度应避免疲劳，保证接班人员充分休息。

航行值班：航次前应研究资料并制定航线，启航前核对。值班期间使用一切助航仪器核对船位航向航速。值班驾驶员应熟悉所有安全航行设备。不应被分配妨碍安全航行的职责。使用雷达时遵守避碰规则。需要时毫不犹豫使用舵、主机和声光信号。负责值班驾驶员应在驾驶台坚持值班，正式交班前不得离开。即使船长在驾驶台，直到船长明确接手责任。接班驾驶员如不能有效履职，不应交班并通知船长。值班期间做好记录，采取预防措施防止海洋污染。

锚泊值班：应足够频繁检查锚位，通过固定航标或物标定位；定期巡查，显示正确号灯号型，鸣放声号；主机备便；能见度不良时及时呼叫船长；如走锚立即抛另一锚并呼叫船长；遵守防污法规。

值班程序考虑因素：天气、能见度、通航密度、临近航行危险、分道通航区域、船员技能经验、助航设备、公司规定与船长命令、港口指令及岸基支持等。

保持正规航行值班决定了驾驶台班组的基本构成，值班驾驶员负责驾驶台管理和人员管理，保持与机舱及其他操作区域通讯联络是驾驶台班组的重要职责。

货物操作 · 稳性强度 · 安全管理
专题三：计划并确保安全地装货、积载、系固、航行中照管货物和卸货

收到航次指令后，船上需进行准备工作：核算航次载货能力、研究货物特性、核算吃水稳性吃水差强度、编制配载图。

载货能力：包括载重能力（净载重量）、容量能力（散装容积）及其他载货能力（杂货船/集装箱船等）。航次净载重量计算：甲板载集装箱时净载重量=总载重量−航次储备量−船舶常数−必要压载水/固定压载；油化船则减去油底等。

总载重量与储备量：舷外水密度一定时，总载重量随最大平均吃水变化。航线水深足够时受载重线限制，船舶航经不同载重线区带时按较低载重线确定总载重量。固定储备量（粮食、供应品、备品等）取定值；可变储备量（燃油、淡水）根据航程及停泊时间确定。

船舶常数：船舶改造、修理、废弃物、海生物附着等重量变化，需测定准确常数值。

强度、吃水差与稳性：装货前配载计划应确保总强度（纵向、横向、扭转）和局部强度，避免船体永久变形。吃水差影响操纵性、快速性：满载时适当范围-0.3m~-0.5m，半载-0.6m~-0.8m，轻载-0.9m~-1.0m，空载＞-2.5%LAP。稳性是船舶倾倒后恢复的能力，航行中应具有适当稳性。

货物特性：大副须研究MSDS，根据不同类型货物确定装卸顺序、装卸要领、航行保管要领及应急反应程序，保证货物安全抵达，无货损货差。

货物系固：根据SOLAS公约，国际航行船舶必须配备经批准的《国际系固手册》。组件货物（件杂货、滚装货、集装箱、大件、木材货）需有效系固。普通件杂货采用填塞支撑，大件货物使用专用系固器具。

隔离要求：危险货物隔离分为四个等级：隔离1（远离）、隔离2（隔离）、隔离3（用整个舱室隔离）、隔离4（用整个舱室纵向隔离）。

货舱通风：自然通风、机械通风、干燥通风。原则：降低舱内露点防止汗水；降低温度防止货物自燃；提供新鲜空气防止腐败；排除有害气体。

稳性衡准（IMO完整稳性七项要求）：
①初稳性高度GM≥0.15m；②复原力臂曲线0°~30°间面积≥0.055M·rad；③0°~40°或进水角较小之间面积≥0.090M·rad；④复原力臂≥0.20M；⑤30°~40°或进水角较大之间面积≥0.030M·rad；⑥最大复原力臂对应角最好≥30°至少25°；⑦满足衡准要求。

强度校核：普通货船根据装载计划确定单位面积实际负荷量及集中载荷限制。

封闭舱室进出安全：封闭舱室（货舱、油舱、压载舱、水舱、燃油舱等）进入前必须测氧（＞21%）、测爆（可燃气体＜1%爆炸下限）、测毒（毒气含量达标）。

卸货作业：大副制定详细卸货计划，报船长审核。卸货过程中按顺序均衡作业，避免中垂或中拱过大；注意稳性及吃水差。卸货设备不得超限，带病工作。卸货后全面检查船体、甲板、舱盖、管线等，发现损坏污染立即报告。

大副批注与货运单证：大副批注是对货运单证的附加说明。重要货运单证包括空距报告、样品化验报告、样品证明、作业时间表、配载图，以及干舱证明、空舱证明、装货计划、卸货计划、货物特性等。

应急 · 保安 · 防污染
专题四：监督和控制法定要求的遵守以及确保海上人命安全、保安与海洋环境保护的措施

由于海上救助条件有限，附近船舶间救助更为及时有效，全体船员须加强应急训练。

遇险信号识别：视觉遇险信号：火箭降落伞信号、橙色烟雾信号、手持火焰信号、信号镜。无线电遇险信号：VHF/MF/HF 数字选择性呼叫、SART、EPIRB。收到遇险警报时应立即通知船长，减速慢行，确定遇险位置，备妥消防救生医疗器材，做好救助准备。

船上紧急情况应急计划整体系统：根据SOLAS及IMO《船上紧急情况应急计划整体系统构成指南》，应急计划编制原则：1）应急反应快速响应，减少损失；2）规定协调、联系方式、船岸联系和程序评价；3）计划、准备和培训：定期培训考核，关键人物了解计划并履行职责；4）反应行动：针对航行、停泊、锚泊、港内等各种紧急情况提供最佳行动指南；5）保证24h通信联系，联系方式及时更新；6）提高船员判定和操作能力。

损害控制计划：针对火灾、爆炸、碰撞等不同损害制定应对措施，包括报警、驾驶台行为、组织行动、应急措施、报告记录等，挽救生命财产和保护海洋环境。

公共安全事件与保安：保安应急小组报告公共事件性质和发展趋势，向沿岸国和港口主管机关通报，提高保安等级。船长应劝阻外来人员登船，集中限制在生活区外。高危险海域航行时应增加雷达开启，船尾强光照明，消防泵出水等防海盗措施。

船舶油污应急计划（SOPEP）

国际航行船舶必须配备油污应急计划，并按计划进行演习。主要内容：
1. 油污事故报告程序（实际排油或可能排油时向最近沿岸国报告）；
2. 需要联系的单位或个人（沿岸国联系人、港口国联系人、船舶安全联系人如船公司、保险公司等）；
3. 减少和控制排油措施（操作性溢油：管系泄漏、舱柜满溢、船体泄漏；海损事故溢油：搁浅、火灾、爆炸、碰撞、船壳破损及严重横倾等）；
4. 船舶与沿岸国、地方当局联系的程序和要求。
另外，SOPEP还应提供优先措施指导（稳性、压力影响、减载等）。

污染事故报告内容：按IMO A.851(20)决议要求，报告船上货物和燃油数量、种类、损坏和污染概况（溢油量、天气、海况、船舶资料等）。

大副需监督防止任何形式海洋污染，严格执行垃圾管理及MARPOL公约要求，做好相关记录簿保管。
